###Rule
1. 시간 약속
	- 데일리스크럼 시간 : 10시 (학습활동있는 날은 스킵)
	- 프로젝트는 10시까지만 진행
2. 짝 프로그래밍을 기본으로 하되 고민되는 사항은 의견 나누기
3. 서로 모르는게 있으면 그 즉시 묻기
4. commit 스타일 : Karma
5. commit 단위: 기능(=함수)단위
6. PR Deadline
	- STEP1은 화요일 저녁 10시까지 어떻게든 PR을 작성한다
	- STEP2는 STEP1 진행도를 보고 이후 논의한다
